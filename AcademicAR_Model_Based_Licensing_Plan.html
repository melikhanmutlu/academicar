Purchase and License Wallet / Satın Alma ve Lisans Cüzdanı

Hybrid flow / Hibrit akış
01

Create / Oluştur

The user creates an account, then creates a publication, article, poster, thesis, or project title.

Kullanıcı hesap açar, ardından yayın, makale, poster, tez veya proje başlığı oluşturur.

02

Upload / Yükle

The publication page directly presents model upload, followed by Free Access, Pro License, and Lifetime License choices.

Yayın sayfasında doğrudan model upload alanı ve Free Access, Pro License, Lifetime License seçenekleri gösterilir.

03

Buy / Satın Al

The user can buy 2 Pro licenses and 3 Lifetime licenses in a single payment, creating license credits.

Kullanıcı tek ödeme ile 2 Pro lisans ve 3 Lifetime lisans alabilir; bunlar lisans kredisi olur.

04

Assign / Ata

Unused licenses appear in the License Wallet and can be assigned to any eligible model later.

Kullanılmamış lisanslar License Wallet içinde görünür ve sonradan uygun modellere atanabilir.

English Positioning

The account is free. The model is the monetized asset. The QR code and viewer URL stay stable while the license behind the model changes.

Türkçe Konumlandırma

Hesap ücretsizdir. Ücretlendirilen varlık modeldir. QR kodu ve viewer URL'i sabit kalır; modelin arkasındaki lisans değişir.

Upgrade Behavior / Yükseltme Davranışı

Stable link and QR / Sabit link ve QR

When a License Is Assigned / Lisans Atandığında

English

  • The model viewer link does not change.
  • The QR code does not change.
  • The model license level and viewer features change.
  • The access duration and license rights are written to the model record.
  • Free-to-Pro and Free-to-Lifetime upgrades reactivate the same public model destination.

Türkçe

  • Model viewer linki değişmez.
  • QR kodu değişmez.
  • Model lisans seviyesi ve viewer özellikleri değişir.
  • Erişim süresi ve lisans hakları model kaydına yazılır.
  • Free'den Pro'ya veya Lifetime'a geçiş aynı public model hedefini yeniden aktif eder.

File Size Edge Case / Dosya Boyutu Senaryosu

English

  • If a model was originally uploaded under Free Access, it may have used the lower Free file size limit.
  • If the user later upgrades to Lifetime, Lifetime features unlock immediately.
  • The user may optionally re-upload a higher quality file to benefit from the 250 MB Lifetime limit.
  • If a Pro or Lifetime license is assigned before upload, that license limit applies from the start.

Türkçe

  • Model ilk olarak Free Access ile yüklendiyse düşük Free dosya limitini kullanmış olabilir.
  • Kullanıcı sonradan Lifetime'a yükseltirse Lifetime özellikleri hemen açılır.
  • Kullanıcı isterse 250 MB Lifetime limitinden faydalanmak için daha kaliteli dosyayı yeniden yükleyebilir.
  • Pro veya Lifetime lisans upload öncesi atanırsa ilgili limit baştan uygulanır.

Institutional Licensing / Kurumsal Lisanslama

B2B offer-based sales / Teklif bazlı satış

English

  • Institutional licensing does not need to be sold as a self-service web checkout in the first version.
  • It can be positioned as "Contact for institutional licensing".
  • Institutional packages may include higher model limits, bulk license credits, multi-user lab accounts, special invoices, multi-model viewers, custom embed solutions, and manual technical support or model optimization services.
  • Large model limits can be handled selectively for B2B customers rather than exposed broadly on the public pricing page.

Türkçe

  • Kurumsal lisanslamanın ilk sürümde self-servis web checkout olarak satılması gerekmez.
  • "Contact for institutional licensing" şeklinde konumlandırılabilir.
  • Kurumsal paketlerde daha yüksek model limitleri, toplu lisans kredileri, çok kullanıcılı lab hesapları, özel fatura, çoklu model viewer, özel embed çözümleri ve manuel teknik destek/model optimizasyon hizmetleri olabilir.
  • Büyük model limitleri herkese açık pricing sayfasında verilmek yerine B2B müşteriler için seçici şekilde yönetilebilir.

Technical Assumptions / Teknik Varsayımlar

Data model direction / Veri modeli yönü

Model-Level License State / Model Seviyesinde Lisans Durumu

English

  • User plan stops being the main business model.
  • Each model stores its license state: free, pro, or lifetime.
  • Each model stores license status, license start, license expiration, and maximum file size.
  • Viewer access control reads the model license instead of the publication package.

Türkçe

  • User plan ana iş modeli olmaktan çıkar.
  • Her model kendi lisans durumunu tutar: free, pro veya lifetime.
  • Her model lisans statüsü, başlangıç, bitiş ve maksimum dosya boyutunu tutar.
  • Viewer erişim kontrolü yayın paketinden değil model lisansından okunur.

License Inventory and Payments / Lisans Envanteri ve Ödemeler

English

  • Unused purchases are stored as license inventory.
  • License credits support unused, assigned, refunded, and expired statuses.
  • A license credit can later be assigned to a model.
  • Payments support line items so one checkout can include multiple Pro and Lifetime licenses.

Türkçe

  • Kullanılmamış satın almalar lisans envanteri olarak saklanır.
  • Lisans kredileri unused, assigned, refunded ve expired statülerini destekler.
  • Bir lisans kredisi sonradan modele atanabilir.
  • Ödemeler satır kalemlerini destekler; tek checkout birden fazla Pro ve Lifetime lisans içerebilir.

Acceptance Criteria / Kabul Kriterleri

Launch checks / Yayına çıkış kontrolleri

English

  • A user can create a free account and create a publication.
  • The publication model upload screen shows Free Access, Pro License, and Lifetime License options.
  • A user can upload a free model and use it for 3 days.
  • When Free Access expires, the QR and viewer link remain the same but access falls back to an upgrade screen.
  • A user can buy 2 Pro licenses and 3 Lifetime licenses in one payment and see them in the account.
  • A user can assign an unused license to any eligible model.
  • Pro and Lifetime upgrades preserve the viewer link and QR code.
  • Lifetime models unlock annotation, saved camera angles, info boxes, guided views, and preview video features.

Türkçe

  • Kullanıcı ücretsiz hesap oluşturup yayın oluşturabilir.
  • Yayın model upload ekranında Free Access, Pro License ve Lifetime License seçenekleri görünür.
  • Kullanıcı free model yükleyip 3 gün kullanabilir.
  • Free Access süresi dolduğunda QR ve viewer linki aynı kalır, erişim upgrade ekranına düşer.
  • Kullanıcı tek ödeme ile 2 Pro lisans ve 3 Lifetime lisans satın alıp hesabında görebilir.
  • Kullanıcı kullanılmamış lisansı uygun herhangi bir modele atayabilir.
  • Pro ve Lifetime yükseltmeleri viewer linkini ve QR kodunu korur.
  • Lifetime modellerde annotation, kaydedilmiş kamera açıları, info box, guided view ve preview video özellikleri açılır.
Implementation note / Uygulama notu: The first implementation should keep the upload and purchase flow simple: allow Free Access upload immediately, allow pre-purchased Pro/Lifetime license assignment before upload for higher file limits, and preserve the same model URL/QR across every upgrade path.
